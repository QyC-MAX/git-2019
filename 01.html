• cd ~ 家目录
  • git config --global user.name：设置用户名全局
  • git config --global user.email：设置邮箱全局
  • cd ~/gitconfig 查看全局用户名邮箱
  • cd .git/config 查看当前用户名邮箱
  • git init：初始化仓库
  • git status： 查看状态
  • git add：提交到暂存区
  • git commit -m ：提交到本地仓库
  • git log：查看所有
  • git checkout file
  • git rm --cached
  • git reset --hard commitID
    分支命令
  1. 查看分支: git branch
  2. 创建分支: git branch 分支名称
  3. 切换分支: git checkout 分支名称
  4. 合并分支: git merge 来源分支名称
  5. 删除分支: git branch -d 分支名称(分支合并后才能被删除)(-D 强制删除)
  6. git stash 储存临时改动
  7. git stash pop : 恢复
