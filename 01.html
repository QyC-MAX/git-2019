• cd ~ 家目录
  • git config --global user.name：设置用户名全局
  • git config --global user.email：设置邮箱全局
  • cd ~/gitconfig 查看全局用户名邮箱
  • cd .git/config 查看当前用户名邮箱
  • git init：初始化仓库
  • git status： 查看状态
  • git add：提交到暂存区
  • git commit -m ：提交到本地仓库
  • git log：查看提交的记录
  • git checkout file：用暂存区中的文件覆盖当前的文件
  • git rm --cached file： 删除暂存区中的文件
  • git reset --hard id 恢复到某一次提交的文件，这次之后的提交的文件会被清除
    分支命令
  1. 查看分支: git branch
  2. 创建分支: git branch 分支名称
  3. 切换分支: git checkout 分支名称
  4. 注意: 切换分支前一定要把文件提交到仓库后在切换分支，不然当前分支的文件会显示到切换的分支上，会造成文件混乱
  5. 合并分支: git merge 被合并的分支 合并完后的分支依然存在
  6. 删除分支: git branch -d 分支名称(分支合并后才能被删除)(-D 强制删除)
  7. git stash 储存临时改动
  8. git stash pop : 恢复
